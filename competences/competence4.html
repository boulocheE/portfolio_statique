Projet 1 : Base de données pour un comité d'entreprise

Présentation du projet :

Une entreprise parisienne est répartie en plusieurs établissements, implantés sur des communes distinctes. Ils disposent d'un Comité d'Entreprise (CE), avec un service enfant. Ces derniers gèrent par exemple les cadeaux de NOël et les vacances.
Nous avons créé une base de données qui permet au comité de gérer les différents cadeaux, et la distribution de ces derniers aux enfants.

Langage utilisé :

  • PostgreSQL
